# FreeRTOS-Seat-Heater-Control-System-for-Tiva-C
 Implementation of an automotive seat heater system for Tiva C using FreeRTOS. Features independent driver/passenger control with three heating intensities (Low/Medium/High) and real-time ADC temperature monitoring. Includes event-driven tasks, sensor diagnostics (5°C-40°C), UART status reporting, and manual GPTM runtime measurements for CPU load.
